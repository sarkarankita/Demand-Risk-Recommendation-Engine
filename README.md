# Supply Chain Demand & Risk Recommendation Engine

## Overview
This project builds a decision-support system to analyze supply chain demand trends,
identify late delivery risks, and generate cost-aware business recommendations.

## Key Features
- Demand trend analysis (time series)
- Late delivery risk monitoring
- Forecasting
- Business recommendation engine
- Interactive Streamlit app

## Tech Stack
- Python
- Pandas, NumPy
- SciPy, Statsmodels
- Scikit-learn
- Streamlit

## Goal
To help operations and business teams take proactive actions
to reduce late deliveries while balancing cost and service levels.
